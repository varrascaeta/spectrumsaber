# SpectrumSaber Project

# ==========================

This is the SpectrumSaber project, a web application for managing and analyzing data from the CONAE's Field Spectroscopy campaigns. The project is built using Python and Django, and it uses Docker for containerization and deployment.

## Installation

Read the [setup instructions](docs/setup.md) for detailed steps on how to set up the project on your local machine.
